# vscode-idiomatic-css-comments-snippets

> VSCode snippets for [idiomatic CSS comments](https://github.com/necolas/idiomatic-css#comments)

## Section `com-section`
```css
/* ==========================================================================
   Section
   ========================================================================== */
```

## Sub-section `com-subsection`
```css
/*
   Sub-section
   ========================================================================== */
```

## Long `com-long`
```css
/**
 * Long
 */
```

## Basic `com-basic`
```css
/* Basic */
```
